# HK Arbitration Plugin

**Hong Kong arbitration law — Arbitration Ordinance (Cap 609), UNCITRAL Model Law, HKIAC Rules, interim measures, award enforcement, and court supervision.**

This plugin provides specialised tools for legal practitioners working with Hong Kong arbitration, including agreement review, interim measures, award enforcement (domestic and cross-border), and set-aside proceedings.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on arbitration matters

## Skills

- **[Arbitration Agreement Review](skills/agreement-review/SKILL.md)** — Review and draft arbitration agreements for compliance with Cap 609, Model Law, and institutional requirements
- **[Interim Measures](skills/interim-measures/SKILL.md)** — Guide to interim measures from HK courts and arbitral tribunals, including cross-border measures under the 2019 PRC Arrangement
- **[Award Enforcement in HK](skills/award-enforcement-hk/SKILL.md)** — Enforce domestic, Model Law, and foreign arbitral awards in Hong Kong
- **[Award Enforcement in China](skills/award-enforcement-china/SKILL.md)** — Enforce HK arbitral awards in Mainland China under the Arrangement
- **[Set-Aside](skills/set-aside/SKILL.md)** — Grounds and procedure for setting aside awards under s.81 of Cap 609

## MCP Servers

| Server | Description |
|--------|-------------|
| HKLII | Hong Kong arbitration case law — set-aside, enforcement, interim measures |
| e-Legislation | Official Arbitration Ordinance (Cap 609) and Model Law |
| HKIAC | HKIAC Rules, Practice Notes, fee calculator |
| PRC Supreme People's Court | SPC interpretations on arbitration enforcement |
