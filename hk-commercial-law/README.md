# Hong Kong Commercial Law Plugin

A Claude plugin providing expertise on Hong Kong commercial law, including contract law, company law, banking and finance, and cross-border commercial transactions in Hong Kong.

## Structure

- `.claude-plugin/` - Plugin configuration and instructions
- `skills/` - Supporting skill definitions
- `data/` - Static reference data
- `hooks/` - Lifecycle hooks

## Usage

Configure in your Claude project to enable Hong Kong commercial law expertise.
