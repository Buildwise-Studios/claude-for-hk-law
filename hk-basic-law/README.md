# HK Basic Law Plugin

**Hong Kong Basic Law — constitutional research, NPCSC interpretations, national security law, and fundamental rights analysis.**

This plugin provides specialised tools for legal practitioners working with the Hong Kong Basic Law, including constitutional review, NSL compliance, and Basic Law interpretation methodology.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on Basic Law matters

## Skills

- **[Constitutional Review](skills/constitutional-review/SKILL.md)** — Analyze Basic Law issues, NPCSC interpretations, and fundamental rights questions
- **[NSL Compliance](skills/nsl-compliance/SKILL.md)** — National Security Law compliance checks for legal practitioners
- **[Basic Law Interpretation](skills/bl-interpretation/SKILL.md)** — Guide to Basic Law interpretation methodology including Article 158

## MCP Servers

| Server | Description |
|--------|-------------|
| HKLII | Hong Kong Legal Information Institute — case law, legislation, BL materials |
| e-Legislation | Official Hong Kong legislation database |
| Westlaw Asia | HK constitutional law commentary and journal articles |
