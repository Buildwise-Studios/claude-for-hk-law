# HK Shipping & Maritime Plugin

**Hong Kong maritime law — admiralty, bills of lading, charterparties, ship arrest, marine insurance, and maritime arbitration.**

This plugin provides specialised tools for maritime law practitioners working with Hong Kong shipping and admiralty matters under Cap 440, Cap 415, admiralty jurisdiction, and international conventions.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on maritime law matters

## Skills

- **[Admiralty Claim](skills/admiralty-claim/SKILL.md)** — Admiralty claim procedure in the Court of First Instance
- **[Bill of Lading](skills/bill-of-lading/SKILL.md)** — Bill of lading review under Cap 440 and Hague-Visby Rules
- **[Charterparty Review](skills/charterparty-review/SKILL.md)** — Charterparty review under Hong Kong law
- **[Ship Arrest](skills/ship-arrest/SKILL.md)** — Ship arrest procedure in Hong Kong waters
- **[Marine Insurance](skills/marine-insurance/SKILL.md)** — Marine insurance policy review under Cap 329

## MCP Servers
