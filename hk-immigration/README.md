# HK Immigration Plugin

**Hong Kong immigration law — right of abode, visa schemes, deportation, and immigration appeals under Cap 115.**

This plugin provides specialised tools for immigration law practitioners working with Hong Kong immigration matters under the Immigration Ordinance (Cap 115) and related admission schemes.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on immigration law matters

## Skills

- **[Employment Visa](skills/visa-employment/SKILL.md)** — Employment visa application review under GEP, SLS, and related schemes
- **[Right of Abode](skills/right-of-abode/SKILL.md)** — Right of abode analysis under Basic Law Art 24 and Part VII Cap 115
- **[Deportation Defense](skills/deportation-defense/SKILL.md)** — Deportation risk assessment and defense under ss.20–25 Cap 115
- **[Top Talent Pass](skills/top-talent-pass/SKILL.md)** — Top Talent Pass Scheme eligibility and application
- **[Immigration Appeal](skills/immigration-appeal/SKILL.md)** — Immigration Tribunal appeal procedure and grounds

## MCP Servers

| Server | Description |

## Related Plugins
