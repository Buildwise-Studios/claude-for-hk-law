# HK Data Privacy Plugin

**Hong Kong data privacy law — PDPO (Cap 486), data protection principles, direct marketing, data breach response, cross-border transfers, and PCPD compliance.**

This plugin provides specialised tools for data privacy practitioners working with Hong Kong's Personal Data (Privacy) Ordinance, including privacy impact assessments, DSAR handling, direct marketing compliance, breach response, and cross-border data transfer analysis.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on data privacy matters

## Skills

- **[PIA (HK)](skills/pia-hk/SKILL.md)** — Privacy impact assessment under the PDPO framework
- **[DSAR (HK)](skills/dsar-hk/SKILL.md)** — Data subject access request handling (40-day timeline)
- **[Direct Marketing](skills/direct-marketing/SKILL.md)** — Direct marketing compliance checklist under s.35A-J
- **[Data Breach (HK)](skills/data-breach-hk/SKILL.md)** — Data breach response plan specific to HK
- **[Cross-Border Transfer](skills/cross-border-transfer/SKILL.md)** — Cross-border data transfer analysis and contractual approach
- **[DPA Review (HK)](skills/dpa-hk/SKILL.md)** — Data processing agreement review under the PDPO
