# HK Property & Conveyancing Plugin

**Hong Kong property law — conveyancing, tenancies, land registration, building management, and stamp duty under Cap 219, Cap 128, and Cap 344.**

This plugin provides specialised tools for property law practitioners working with Hong Kong real estate matters, including sale and purchase transactions, leasing, land registration, building management disputes, and stamp duty calculations.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on property law matters

## Skills

- **[SPA Review](skills/spa-review/SKILL.md)** — Review and advise on formal sale and purchase agreements for Hong Kong property
- **[Tenancy Review](skills/tenancy-review/SKILL.md)** — Review residential and commercial tenancy agreements under Cap 7
- **[Land Search Guide](skills/land-search-guide/SKILL.md)** — Guide to Land Registry searches and title investigation
- **[Building Management](skills/building-management/SKILL.md)** — DMC review, IO incorporation, building management disputes
- **[Stamp Duty Calculator](skills/stamp-duty-calc/SKILL.md)** — Calculate stamp duty including AVD, BSD, SSD, and NRSD

## MCP Servers

| Server | Description |
|--------|-------------|
| Land Registry | Search HK land records, register deeds and charges |
| RVD | Property valuations, rates, government rent |
| HKLII | Property law case law |
| e-Legislation | Conveyancing and Property Ordinance, Land Registration Ordinance |
