# HK Intellectual Property Plugin

**Hong Kong intellectual property law — trade marks (Cap 559), patents (Cap 514), copyright (Cap 528), registered designs (Cap 522), passing off, fair dealing, and cross-border IP issues.**

This plugin provides specialised tools for legal practitioners working with Hong Kong intellectual property matters, including clearance searches, infringement analysis, fair dealing assessment, and takedown procedures.

## Quick Start

1. Install this plugin in your Claude Code environment
2. Configure your MCP server connections (optional — see `.mcp.json`)
3. Use the skills below to research and draft on IP matters

## Skills

- **[Trade Mark Clearance](skills/tm-clearance/SKILL.md)** — Trade mark availability and clearance searches via IPD, risk assessment, and filing strategy
- **[Trade Mark Infringement](skills/tm-infringement/SKILL.md)** — Trade mark infringement and passing off analysis, remedies, and enforcement
- **[Patent Review](skills/patent-review/SKILL.md)** — Patent validity and infringement analysis for standard and short-term patents
- **[Copyright Fair Dealing](skills/copyright-fair-dealing/SKILL.md)** — Fair dealing analysis under Cap 528, permitted purposes, and limitations
- **[Copyright Takedown](skills/takedown-528/SKILL.md)** — Copyright takedown procedures under Cap 528 ss.88–91 for online service providers

## MCP Servers

| Server | Description |
|--------|-------------|
| IPD | Hong Kong Intellectual Property Department — trade mark, patent, design search |
| HKLII | Hong Kong IP case law including trade mark infringement, passing off, copyright |
| e-Legislation | Official Hong Kong IP ordinances |
| Customs & Excise | Border measures and criminal enforcement information |
