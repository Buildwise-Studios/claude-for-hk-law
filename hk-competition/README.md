# Hong Kong Competition Law Plugin

A Claude plugin providing expertise on Hong Kong competition law, including the Competition Ordinance (Cap. 619), merger control, non-compliance, enforcement, and related regulatory matters.

## Structure

- `.claude-plugin/` - Plugin configuration and instructions
- `skills/` - Supporting skill definitions
- `data/` - Static reference data
- `hooks/` - Lifecycle hooks

## Usage

Configure in your Claude project to enable Hong Kong competition law expertise.
